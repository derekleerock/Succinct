UITableView

extension UITableView
  • findView(satisfyingCondition:)

    Searches through all rows of all sections as well as all header views within a UITableView for a view that satisfies the passed in condition.

    Note

    Once an object is found matching the provided criteria the remainder of the view hierarchy is not searched.

    Declaration

    Swift

    @objc
public func findView(satisfyingCondition: UIViewEvaluator) -> UIView?

    Parameters

    satisfyingCondition

    A UIViewEvaluator closure that takes a UIView as an argument and evaluates it against some kind of condition and returns an optional UIView if a view is found that matches the condition.

    Return Value

    An optional UIView, if one is found.
