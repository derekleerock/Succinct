  • selectedTab(isAKindOf:)

    Checks if the UITabBarController’s selectedViewController property matches the type passed in.

    Note

    This API method is helpful to confirm the type of UIViewController object without having to be coupled to whether or not the UIViewController is inside a UINavigationController or not.

    Declaration

    Swift

    public func selectedTab<T>(isAKindOf viewControllerType: T.Type) -> Bool where T : UIViewController

    Parameters

    viewControllerType

    Type of class to compare to the UITabBarController’s selectedViewController property.

    Return Value

    A boolean value indicating if the selected tab’s class matches or not.
